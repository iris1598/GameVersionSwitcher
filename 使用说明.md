# 游戏版本切换器 使用说明

## 功能简介

为同一游戏的两个服务器版本（例如国服/外服、官服/B服）提供一键切换启动。
启动某个版本时，会自动把该版本的差异文件覆盖到游戏目录，然后启动游戏 exe。

## 使用步骤

### 1. 准备差异文件

把每个版本相对游戏的差异文件，按 **与游戏目录相同的相对结构** 放到两个独立文件夹里。

例如游戏目录是 `D:\Games\MyGame`，差异文件包括 `config.ini` 和 `data\server.dat`，那么版本A的差异文件夹应该是：

```
D:\DiffA\
  ├── config.ini
  └── data\
       └── server.dat
```

程序会把这些文件按相对路径复制（覆盖）到 `D:\Games\MyGame` 下。

### 2. 配置

双击 `游戏版本切换器.exe` 打开主界面，填入：

- **版本A名称** / **版本B名称**：自定义，会作为快捷方式的名字
- **版本A/B 差异文件路径**：对应版本的差异文件所在目录
- **游戏目录**：游戏安装目录
- **游戏 exe**：游戏启动程序（如 `game.exe`）

点 **保存配置**，配置会保存到 exe 同目录的 `config.json`。

### 3. 创建桌面快捷方式

点 **创建A快捷方式** / **创建B快捷方式**，会在桌面生成对应的 `.lnk` 快捷方式。

之后直接双击桌面快捷方式即可：自动复制差异文件 + 启动游戏。无需再打开主界面。

### 4. 直接启动

也可以在主界面直接点 **▶ 启动 A** / **▶ 启动 B**，效果与快捷方式相同。

## 工作原理

1. 启动某版本时，递归遍历对应差异文件目录下的所有文件。
2. 按相对路径覆盖到游戏目录（保留原有目录结构，必要时创建子目录）。
3. 文件复制完成后，启动游戏 exe，程序自身退出。

## 注意事项

- **复制是覆盖式**：会直接覆盖游戏目录中同名文件。建议第一次使用前备份游戏目录。
- **不会清理**：如果版本A的某个文件在版本B中不存在，切换到版本B时该文件会保留。请确保差异文件目录只包含真正需要切换的文件。
- **首次以管理员身份运行**：如果游戏目录在 `C:\Program Files` 等需要管理员权限的位置，请右键 → 以管理员身份运行本程序或对应快捷方式。
- **杀毒软件误报**：PyInstaller 打包的单文件 exe 可能被部分杀毒软件误报，可加入白名单。

## 文件清单

| 文件 | 说明 |
|------|------|
| `游戏版本切换器.exe` | 主程序 |
| `config.json` | 配置文件（首次保存后自动生成，与 exe 同目录） |
| `桌面/版本A.lnk` `桌面/版本B.lnk` | 桌面快捷方式（手动创建） |
| `game_launcher.py` | 源代码（可选，仅用于二次开发） |
| `使用说明.md` | 本文件 |
